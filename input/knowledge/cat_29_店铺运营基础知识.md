# 店铺运营基础知识

数据来源：Amazon Global Selling 官方资料

## Seller Central 后台核心模块

| 模块 | 功能 | 使用频率 |
|------|------|---------|
| Catalog（商品目录） | 添加/编辑产品 | 上新时 |
| Inventory（库存） | 管理 FBA/FBM 库存 | 每日 |
| Pricing（定价） | 价格设置、自动定价 | 每周 |
| Orders（订单） | 订单管理、退货处理 | 每日 |
| Advertising（广告） | PPC 广告管理 | 每日 |
| Reports（报告） | 业务/广告/库存报告 | 每周 |
| Performance（绩效） | 账户健康指标 | 每周 |
| Brand（品牌） | A+、旗舰店、品牌分析 | 按需 |

## Buy Box（黄金购物车）
- 约 83% 的销售发生在 Buy Box
- 影响因素：价格、FBA、账户绩效、库存
- FBA 卖家比 FBM 更容易获得
- 竞争 Buy Box 的核心：有竞争力的配送总价

## 亚马逊搜索排名因素
1. 关键词相关性（标题 > 五点 > 描述 > Search Terms）
2. 销售速度（Sales Velocity）
3. 转化率（CVR）
4. 评论数量和星级
5. 库存状态（有货 > 缺货）
6. 价格竞争力

## 账户绩效管理
- 每日检查 Account Health Dashboard
- ODR < 1%（订单缺陷率）
- 迟发率 < 4%
- 有效追踪率 > 95%
- 政策违规：及时申诉（POA 行动计划）

## 新手运营常见误区
- ❌ 频繁修改 Listing（会影响排名稳定性）
- ❌ 一出单就涨价（破坏转化率）
- ❌ 不做广告等自然流量（新品需广告推动）
- ❌ 忽视客户消息（24 小时内必须回复）
