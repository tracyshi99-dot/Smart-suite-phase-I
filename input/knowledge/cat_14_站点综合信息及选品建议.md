## 多站点运营策略
- 同一账户可开通多站点（北美联合账户、欧洲联合账户）
- 欧洲五站共用 FBA 库存（EFN/Pan-EU 物流）
- 不同站点的 Listing 需本地化翻译
- 定价需考虑当地竞争和消费水平

## 各站点选品差异化建议
- 美国：竞争激烈，需差异化或品牌溢价
- 欧洲：注重合规（CE认证、WEEE、EPR）
- 日本：品质要求极高，包装精致
- 中东：斋月/开斋节是旺季，宗教文化敏感
- 澳洲：户外运动需求大，季节相反

## 新手选站建议
1. 预算充足 → 美国站起步
2. 英语好 → 美国+英国联动
3. 产品有认证 → 考虑欧洲站
4. 小众精品 → 日本站或澳洲站竞争小
