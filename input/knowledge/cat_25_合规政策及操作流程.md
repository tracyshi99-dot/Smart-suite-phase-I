# 合规政策及操作流程

数据来源：Amazon Global Selling 官方资料

## 亚马逊核心合规政策

| 政策领域 | 核心要求 | 违规后果 |
|---------|---------|---------|
| 知识产权 | 不侵犯商标/专利/版权 | 下架+警告+封号 |
| 产品安全 | 符合目的国安全标准 | 下架+召回+罚款 |
| Listing 合规 | 标题/图片/描述符合规范 | 抑制+下架 |
| 评论政策 | 不得操纵评论/刷单 | 封号 |
| 账户健康 | ODR<1%、迟发率<4% | 限制销售+封号 |

## 产品认证要求（按站点）

| 认证 | 适用站点 | 适用品类 | 说明 |
|------|---------|---------|------|
| CE | 欧洲 | 电子/玩具/机械 | 欧盟安全标志 |
| FCC | 美国 | 电子产品 | 美国联邦通信 |
| UL | 美国 | 电器/电池 | 安全认证 |
| PSE | 日本 | 电子电器 | 日本强制 |
| CPC | 美国 | 儿童产品 | 儿童产品证书 |
| CPSC | 美国 | 消费品 | 消费品安全 |

## EPR（生产者延伸责任）
- 德国：VerpackG（包装法）、ElektroG（电子电器）、BattG（电池）
- 法国：包装、电器、电池、家具、纺织品
- 注册平台：LUCID（德国）、各品类 PRO 组织

## 账户健康指标

| 指标 | 合格标准 | 计算方式 |
|------|---------|---------|
| 订单缺陷率 ODR | < 1% | A-to-Z + 退单 + 差评 |
| 迟发率 LSR | < 4% | 确认发货超时比例 |
| 预取消率 | < 2.5% | 卖家取消订单比例 |
| 有效追踪率 | > 95% | 有物流单号的订单比例 |

## 合规操作建议
- 上架前核查知识产权（商标数据库 TESS/EUIPO）
- 产品测试报告提前准备
- 使用亚马逊合规参考工具（Compliance Reference）
- 定期检查账户健康仪表板
