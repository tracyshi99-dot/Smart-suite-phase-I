# 亚马逊开店成本费用详解

数据来源：Amazon Global Selling 官方资料（2025-2026 费率）

## 基础费用

| 费用类型 | 金额 | 说明 |
|---------|------|------|
| 专业卖家月租 | $39.99/月 | 个人卖家免月租但每件加收 $0.99 |
| 销售佣金 | 8%-17% | 按品类不同（大部分品类 15%） |

## 各品类佣金率一览

| 品类 | 佣金率 | 最低佣金 |
|------|--------|---------|
| 电子产品 | 8% | $0.30 |
| 电脑及配件 | 8% | $0.30 |
| 摄影器材 | 8% | $0.30 |
| 家居厨房 | 15% | $0.30 |
| 运动户外 | 15% | $0.30 |
| 玩具游戏 | 15% | $0.30 |
| 服装鞋靴 | 17% | $0.30 |
| 珠宝首饰 | 20%（$250以下）/5%（$250以上） | $0.30 |

## FBA 费用结构

| 费用项目 | 金额范围 | 计费方式 |
|---------|---------|---------|
| 配送费（标准尺寸） | $3.22-$6.90 | 按重量和尺寸 |
| 配送费（大件） | $9.73-$40+ | 按重量和尺寸 |
| 月度仓储费（1-9月） | $0.87/立方英尺 | 按库存体积 |
| 月度仓储费（10-12月） | $2.40/立方英尺 | 旺季加价 |
| 长期仓储费 | $6.90/立方英尺 | 超 365 天库存 |
| 移除费 | $0.97-$5.00/件 | 退回或销毁 |

## 广告费用参考
- 美国站平均 CPC：$0.75-$1.50
- 建议广告预算：日均 $20-50（新品期）
- 广告占销售额比例：建议 15-25%（新品）、8-12%（成熟产品）

## 隐性成本提醒
- 退货处理费、退款佣金
- 促销折扣（Coupon、Lightning Deal 费用 $150/次）
- 品牌注册费用（免费，但商标注册约 ¥2,000-5,000）
- UPC/EAN 条码购买
