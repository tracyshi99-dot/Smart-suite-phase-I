# 做跨境电商的准备工作

数据来源：Amazon Global Selling 官方资料

## 必备材料清单

| 材料 | 说明 | 注意事项 |
|------|------|---------|
| 营业执照 | 个体/企业均可 | 注册满 1 年以上更易通过 |
| 法人身份证 | 正反面照片 | 需在有效期内 |
| 双币信用卡 | Visa/Mastercard | 用于扣除月租和费用 |
| 收款账户 | 第三方收款工具 | 万里汇、Payoneer、连连等 |
| 联系方式 | 手机号+邮箱 | 建议使用未注册过亚马逊的邮箱 |

## 能力准备
- 基础选品能力：了解市场需求和竞品分析方法
- Listing 撰写能力：标题、五点描述、关键词优化
- 供应链管理：找到可靠的供应商和货代
- 基础财务意识：了解成本结构和利润计算

## 工具准备
- 选品工具：Helium 10、Jungle Scout、卖家精灵
- ERP 系统：积加、领星、马帮
- 收款工具：万里汇（WorldFirst）、Payoneer
- 物流服务商：纵腾、递四方、云途

## 心态准备
- 前 3 个月大概率不盈利，需要耐心
- 持续学习平台规则变化
- 做好长期投入打算（至少 1 年）
- 不要铺货思维，聚焦精品模式
