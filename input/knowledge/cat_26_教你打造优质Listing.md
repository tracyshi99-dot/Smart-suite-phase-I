# 教你打造优质Listing

数据来源：Amazon Global Selling 官方资料

## Listing 核心组成

| 要素 | 作用 | 优化重点 |
|------|------|---------|
| 标题 (Title) | 搜索排名+点击率 | 核心关键词前置，200字符内 |
| 五点描述 (Bullet Points) | 转化率 | 每点突出一个卖点，含关键词 |
| 产品描述 (Description) | 补充信息 | 品牌故事+使用场景 |
| 图片 (Images) | 点击率+转化率 | 主图白底，7张辅图 |
| A+ 页面 | 转化率提升5-10% | 图文并茂，品牌故事 |
| 后台关键词 | 搜索可见性 | 250字节 Search Terms |

## 标题公式
`品牌名 + 核心关键词 + 材质/规格 + 核心卖点 + 适用场景/人群`

示例：BrandX Wireless Bluetooth Earbuds - Active Noise Cancelling, 40H Battery, IPX7 Waterproof - for Running & Gym

## 图片要求
- 主图：纯白背景（RGB 255,255,255），产品占比 85%+
- 辅图建议：场景图、功能图、尺寸对比图、包装内容图、细节图、卖点图
- 尺寸：最长边 2000px+（支持缩放）
- 格式：JPEG/PNG，sRGB 色彩空间

## 关键词优化工具
- Amazon Brand Analytics（品牌分析）
- Helium 10 Cerebro/Magnet
- Amazon Search Terms Report
- 竞品 Listing 反查

## Listing 优化检查清单
- ✅ 标题含 3-5 个核心关键词
- ✅ 五点描述覆盖功能、材质、使用场景、人群、售后
- ✅ 图片 7 张以上，含视频
- ✅ Search Terms 填满，不重复标题词
- ✅ A+ 页面已创建（品牌注册卖家）
- ✅ 价格有竞争力
