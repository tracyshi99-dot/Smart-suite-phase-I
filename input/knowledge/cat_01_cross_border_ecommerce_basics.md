# 跨境电商知识早知道

## 数据来源
亚马逊全球开店官方网站 (https://gs.amazon.cn)

## 核心事实

- 跨境电商是指通过互联网平台将商品销售到其他国家或地区的电子商务模式
- 中国跨境电商出口规模持续增长，已成为外贸新业态的重要组成部分
- 主要模式包括：B2B（企业对企业）、B2C（企业对消费者）、C2C（个人对个人）
- 跨境电商与传统外贸的区别：去中间化、直接触达终端消费者、订单碎片化、物流多样化
- 中国卖家优势：供应链完善、制造成本优势、品类丰富、电商运营经验成熟
