# 欧洲增值税VAT介绍

数据来源：Amazon Global Selling 官方资料

## 什么是 VAT
VAT（Value Added Tax）即增值税，是欧洲对商品和服务征收的消费税。在欧洲站销售必须注册并缴纳 VAT。

## VAT 注册义务触发条件
- 在某国存放库存（使用 FBA 该国仓库）
- 从某国发货给当地买家
- 超过远程销售阈值（EU 统一 €10,000）

## VAT 申报流程
1. 注册各国 VAT 税号（通过税务代理或亚马逊 VAT 服务）
2. 按期申报（月度或季度，各国不同）
3. 缴纳税款 = 销项税 - 进项税
4. 保留发票和记录至少 7 年

## 常见问题
- Q：不注册 VAT 会怎样？A：账户可能被停用，面临补税+罚款
- Q：哪些产品有低税率？A：食品、书籍、儿童用品等（各国不同）
- Q：可以退回进口 VAT 吗？A：可以，通过申报时抵扣
