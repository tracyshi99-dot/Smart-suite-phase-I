# 怎么做跨境电商及流程费用了解

数据来源：Amazon Global Selling 官方资料

## 开店全流程
1. 注册亚马逊卖家账户（1-2 周）
2. 选品与供应链准备（2-4 周）
3. 创建 Listing 并发货至 FBA 仓库（1-2 周）
4. 上线销售 + 广告推广
5. 持续运营优化

## 费用构成一览

| 费用项目 | 金额（美元） | 说明 |
|---------|------------|------|
| 专业卖家月租 | $39.99/月 | 个人卖家免月租但每件收 $0.99 |
| 销售佣金 | 8%-15% | 按品类不同，大部分为 15% |
| FBA 配送费 | $3.22-$10+ | 按尺寸重量计费 |
| FBA 仓储费 | $0.87-$2.40/立方英尺 | 旺季（10-12月）费率更高 |
| 广告费 | 建议占销售额 15-25% | 新品期投入更高 |

## 启动资金建议

| 模式 | 最低投入 | 建议投入 |
|------|---------|---------|
| 轻资产试水 | ¥5,000 | ¥10,000 |
| 正常起步 | ¥20,000 | ¥50,000 |
| 品牌出海 | ¥100,000+ | ¥200,000+ |

## 常见问题
- Q：没有营业执照能开店吗？A：个人卖家可用身份证注册，但建议用企业资质
- Q：多久能回本？A：正常 6-12 个月，取决于品类和运营能力
- Q：必须用 FBA 吗？A：不强制，但 FBA 商品更容易获得 Prime 标志和流量
