# 亚马逊商城基础情况了解

数据来源：Amazon Global Selling 官方资料

## Amazon 基础数据
- 全球站点：20+ 个站点，覆盖北美、欧洲、亚太、中东
- 全球活跃买家：超过 3 亿
- Prime 会员：全球超过 2 亿
- 第三方卖家占比：约 60% 的销售额来自第三方卖家
- 中国卖家占比：约占第三方卖家的 30-40%

## 亚马逊商业模式
| 模式 | 说明 | 适合人群 |
|------|------|---------|
| FBA（亚马逊物流） | 卖家发货至亚马逊仓库，由亚马逊配送 | 大部分卖家 |
| FBM（自发货） | 卖家自行配送给买家 | 大件/定制产品 |
| Vendor（供应商） | 卖给亚马逊，由亚马逊销售 | 大品牌 |

## 核心流量机制
- A9/A10 搜索算法：关键词匹配 + 转化率 + 评论
- 广告系统：Sponsored Products/Brands/Display
- 推荐算法：Frequently Bought Together、Related Products
- Prime Day / 黑五等大促节点流量爆发

## 卖家类型
- 个人卖家：免月租，每件 $0.99，功能受限
- 专业卖家：$39.99/月，可使用广告、品牌工具

## 亚马逊对卖家的核心要求
- 订单缺陷率 (ODR) < 1%
- 迟发率 < 4%
- 取消率 < 2.5%
- 遵守知识产权和产品安全政策
