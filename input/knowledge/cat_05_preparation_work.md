# 做跨境电商的准备工作

## 数据来源
亚马逊全球开店官方网站 (https://gs.amazon.cn)

## 注册前必备材料

| 材料 | 要求 |
|------|------|
| 营业执照 | 中国大陆/香港公司均可，个体工商户亦可 |
| 法人身份证 | 身份证正反面清晰扫描件 |
| 双币信用卡 | Visa/MasterCard，用于支付月租费 |
| 收款账户 | 支持第三方收款（万里汇、派安盈、连连等） |
| 联系方式 | 有效邮箱和手机号码 |
| 公司地址证明 | 对账单或公共事业费账单（部分站点需要） |

## 能力准备

- **资金准备**：至少预留3-6个月运营资金
- **选品能力**：了解目标市场消费者需求和竞争格局
- **运营知识**：熟悉平台规则、Listing优化、广告投放基础
- **供应链资源**：稳定的供应商、样品打版能力
- **物流认知**：了解FBA/FBM模式区别、头程物流方案
- **合规意识**：产品认证（CE/FDA/CPC等）、知识产权、税务

## 建议准备周期

| 阶段 | 时长 | 重点 |
|------|------|------|
| 学习调研 | 1-2周 | 平台规则、行业现状 |
| 材料准备 | 1周 | 营业执照办理、账户开设 |
| 选品确认 | 2-4周 | 市场调研、样品测试 |
| 注册开店 | 1-3天 | 在线提交注册 |
