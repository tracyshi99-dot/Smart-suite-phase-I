# 新手怎么注册亚马逊

数据来源：Amazon Global Selling 官方资料

## 注册方式
- 官方通道：Amazon Global Selling 官网 → 点击"注册"
- 招商经理通道：通过官方招商链接注册（推荐，有专属支持）

## 注册所需材料

| 材料 | 具体要求 | 注意事项 |
|------|---------|---------|
| 营业执照 | 中国大陆/香港公司均可 | 注册满 1 年+，经营范围含贸易 |
| 法人身份证/护照 | 正反面清晰照片 | 有效期内 |
| 双币信用卡 | Visa 或 Mastercard | 确保有效且可扣款 |
| 银行对账单 | 近 3 个月 | 公司户或法人个人户 |
| 联系信息 | 手机号 + 邮箱 | 未注册过亚马逊的邮箱 |
| 收款账户 | 万里汇/Payoneer 等 | 注册后在后台绑定 |

## 注册流程（约 30-60 分钟）
1. 打开 sellercentral.amazon.com → Create Account
2. 填写公司信息（名称、地址、注册号）
3. 填写法人信息（姓名、身份证、住址）
4. 绑定信用卡
5. 完成身份验证（视频/文件验证）
6. 等待审核（通常 1-3 个工作日）

## 视频验证注意事项
- 背景干净，光线充足
- 出示身份证原件 + 营业执照原件
- 全程不超过 5 分钟
- 可能要求展示银行对账单

## 常见拒绝原因及解决方法
- 材料模糊 → 重新拍摄清晰照片
- 信息不一致 → 确保所有材料信息完全一致
- 关联已有账户 → 使用全新网络环境和设备
- 营业执照问题 → 确认执照状态正常且未被其他人使用
