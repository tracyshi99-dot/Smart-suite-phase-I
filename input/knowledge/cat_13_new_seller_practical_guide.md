# 新卖家入门实操宝典

## 数据来源
亚马逊全球开店官方网站 (https://gs.amazon.cn)

## 新卖家第一周行动清单

1. **Day 1-2**：完善账户信息、设置收款方式、绑定信用卡
2. **Day 3-4**：研究目标品类BSR排名，分析竞品Listing
3. **Day 5-6**：确定首款产品，联系供应商获取报价
4. **Day 7**：创建首个Listing（先从1-2个SKU起步）

## Listing创建核心要素

| 要素 | 要求 |
|------|------|
| 标题 | 80-200字符，含核心关键词+品牌+功能+规格 |
| 主图 | 白底、1600×1600px以上、产品占图85%以上 |
| 五点描述 | 突出卖点、包含关键词、解决买家痛点 |
| 产品描述/A+ | 品牌卖家可用A+图文，提升转化率 |
| 后端关键词 | 250字节以内，不重复标题已有词 |
| 价格策略 | 参考竞品前3名定价，新品期可略低10-15% |

## FBA发货流程（首次）

1. 创建FBA发货计划
2. 打印箱唛和FNSKU标签
3. 选择头程物流方式（空运/海运/快递）
4. 按要求包装贴标
5. 发货至指定FBA仓库
6. 等待接收上架（通常3-7天）

## 新卖家常见错误

- ❌ 一次性上架太多SKU（建议从3-5个开始）
- ❌ 备货过多导致资金积压
- ❌ 忽视产品合规认证
- ❌ 完全照搬竞品Listing
- ❌ 不开广告等待自然流量
