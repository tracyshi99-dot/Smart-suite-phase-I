# 开店审核常见问题解答

数据来源：Amazon Global Selling 官方资料

## 审核流程
1. 提交注册资料
2. 系统初审（自动比对，1-3 天）
3. 身份验证（视频验证或明信片验证）
4. 人工复审（如有问题，3-7 天）
5. 审核通过 → 账户激活

## 常见审核问题 FAQ

| 问题 | 解答 |
|------|------|
| 审核需要多长时间？ | 通常 1-3 个工作日，复杂情况最长 2-4 周 |
| 审核不通过能重新提交吗？ | 可以，按拒绝邮件要求补充材料 |
| 个人能注册吗？ | 可以（个人卖家），但建议企业资质 |
| 一个人能注册多个店铺吗？ | 原则上一人/一企业一店，违规关联会封号 |
| 需要实际业务地址吗？ | 是，需与营业执照地址一致 |

## 高频被拒原因及解决

| 被拒原因 | 占比 | 解决方案 |
|---------|------|---------|
| 材料模糊/不清晰 | ~30% | 重新拍摄，确保文字清晰可读 |
| 信息不一致 | ~25% | 核对所有材料的姓名、地址完全一致 |
| 疑似关联 | ~20% | 使用全新设备+网络+邮箱+信用卡 |
| 营业执照异常 | ~15% | 确认工商信息正常，非异常经营 |
| 银行信息问题 | ~10% | 确保对账单为近 3 个月且包含完整信息 |

## 视频验证技巧
- 选择安静、光线充足的环境
- 提前准备好所有原件（身份证 + 营业执照）
- 保持镜头稳定，按审核员指示操作
- 不要紧张，正常沟通即可
- 如果听不懂可要求重复

## 审核通过后第一步
1. 完善店铺信息（Logo、店铺名称）
2. 绑定收款账户
3. 设置税务信息（W-8BEN-E 表格）
4. 了解卖家后台功能模块
