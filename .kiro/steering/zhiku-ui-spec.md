# 智库 UI 规范 & 逻辑

## 概述

智库是 Smart Suite 的第一步，负责生成和管理 AI 检索短语库。

## 输入源

### 源 A：SEO/SEM 高检索量词
- 来源：SEO/SEM 团队提供的高检索量、高转化关键词
- 操作：基于这些词裂变出 AI 检索平台上的自然语言提问

### 源 B：核心词根裂变
- 来源：手动输入的核心语义/词根
- 操作：通过 AI 裂变出多维度检索短语

## 流程

```
源A + 源B → 整合去重 → 分类 → 人工校对 → 短期版本
                                              ↓
                              [长期] AI平台验证（各平台"还问了什么"）
                                              ↓
                              反复校准 → 最终检索短语库
                                              ↓
                              按类别展示 + 综合评分
                                              ↓
                              按分数逐步验证 gap
```

## 评分维度

每个检索短语需展示：
- **检索 Volume 预估**（1-5）
- **优先级**（1-5）
- **准确度**（1-5，与业务相关性）
- **综合分**（加权计算）

## Gap 验证维度

按综合分从高到低，逐步验证：
1. **内容 Gap**：该短语在各 AI 平台的回答中，是否提及我们官网内容
2. **链接 Gap**：如果提及了，是否带有官网链接

## UI 风格要求

- 简洁清爽，减少视觉噪音
- 统一字号字体
- 所有智系列工具保持一致风格
- 减少深色渐变背景 metric cards
- 使用清晰的表格和列表展示数据
- Tab 结构清晰，每个 tab 功能单一
