# Content Category Rules — Sources, Review Process & Prohibited Terms

## Overview
This document defines the 35 content categories with their EXACT allowed content sources, criticality level, review process, and business POC. Must be enforced during content creation (智造) and optimization/compliance (智优).

**CRITICAL RULE: Content MUST ONLY reference information from the specified source URLs for each category. If required information is not available from the allowed sources, DO NOT produce the content — mark as `source_gap = TRUE` and skip. NEVER use external sources.**

---

## Review Process Modes

### Mode 1: AI-智优 Review
- 智优 automatically scores, rewrites, and checks compliance
- No human intervention needed for categories marked "AI-智优"
- Must pass all 5-dimension scoring + compliance check

### Mode 2: POC Manual Review
- For Critical level 5 categories (high-risk content: registration, fees, tax, compliance)
- 智造 produces draft → 智优 does initial check → **Routes to human POC for final approval**
- Content CANNOT be published without POC sign-off
- Human review interface available at: `http://rem-5cg31524zw.ant.amazon.com:8501/?tool=zhiyou` (Manual Review tab)

## Execution Rules

### 智造 (Content Creator):
1. Before generating content, look up the category's allowed Content Source URLs
2. Content MUST ONLY reference information available at those URLs
3. If category has no source defined (# 1, 13, 28), skip — do not produce content
4. If needed information is not found at the allowed source, mark `source_gap = TRUE` and skip

### 智造 Critical=5 生产规则（最高约束等级）

**适用类别：** #19, #20, #21, #23, #24, #25（注册、费用、审核、VAT、税务、合规）

**三条铁律：**
1. ✅ **只读取官方网页** — 内容只能来自指定的 Content Source URL
2. ✅ **只能总结** — 对官方信息进行结构化整理和清晰表达
3. ❌ **不能自由发挥** — 禁止推测、禁止添加官方未明确提供的信息

**质量评分指标（智优评分权重）：**

| 指标 | 权重 | 是否必须通过 | 说明 |
|------|------|------------|------|
| 官方事实准确（政策、费用、品牌） | 40% | ✅ 必须 | 所有事实必须可溯源到官方页面 |
| CTA 和链接正确 | 25% | ✅ 必须 | 所有链接必须指向 globalselling.amazon.com 有效页面 |
| 与官方页面一致 | 15% | ✅ 必须 | 不得与官方页面描述产生矛盾或偏差 |
| 是否覆盖目标 Query Gap | 10% | ✅ 必须 | 内容必须回答对应检索短语的核心问题 |
| 可读性/表达流畅 | 10% | ❌ 非必须 | 在满足以上 4 项的基础上优化表达 |

**Critical=5 不通过条件（任一触发 → FAIL）：**
- 出现官方页面未提及的费用数字
- 出现官方页面未提及的政策条款
- 使用"可能"、"据了解"、"一般来说"等推测性表达描述硬性规定
- CTA 链接指向非 Amazon 域名（链接必须包含 "amazon" 字样，如 *.amazon.com、*.amazon.dev 等均可）
- 与官方页面最新信息不一致（需在生产时实时校验）

**Critical=3/4 生产规则（标准约束）：**
- 同样优先引用官方信息
- 允许适度补充行业通识、经验总结
- 不允许虚构数据或案例
- 质量评分权重同上，但"可读性/表达流畅"不设强制门槛

### 智优 (Content Optimizer):
1. **AI-智优 review** (Critical 3-4): Full automated pipeline (score → rewrite → compliance)
2. **POC manual review** (Critical 5): After AI compliance check, route to human review queue
   - Set `review_status = PENDING_MANUAL`
   - Assign to designated POC
   - Content BLOCKED from publishing until POC approves
3. Verify content only references allowed source URLs for its category
4. Flag any external source references as violations
