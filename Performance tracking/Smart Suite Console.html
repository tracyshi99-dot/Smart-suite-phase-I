🏠 Smart Suite 总览

🧠 Smart Suite 是什么？

Smart Suite 是一套 AI 驱动的 GEO (Generative Engine Optimization) 内容流水线工具， 帮助亚马逊全球开店团队高效生产、优化和分发能够被 AI 搜索引擎（ChatGPT、Perplexity、DeepSeek 等）引用的内容。

📋 Smart Suite 工作原理

keyboard_arrow_right

📚 智库 — 检索短语生成与管理

从 SEO/SEM 关键词或核心词根裂变生成 AI 原生检索短语，自动分类、去重、评分

keyboard_arrow_right

✍️ 智造 — 内容生成

基于选中的检索短语，AI 自动生成 SEO+GEO 双优化的长文章

keyboard_arrow_right

🔧 智优 — 内容优化

一键完成：AI 评分（5维度）→ 重写优化 → 合规审查

keyboard_arrow_right

📦 智布 — 格式化发布

将优化后的内容转换为 JSON 结构化数据和 Word 文档，用于 CMS 发布

keyboard_arrow_right

📡 智传 — 内容分发

将内容分发到各个渠道（官网、第三方平台等），追踪发布状态

keyboard_arrow_right

📈 智析 — 效果分析

追踪 GEO + WW Direct Reg Start 趋势，周度/月度/YTD 数据分析，归因判断

keyboard_arrow_right

🎯 智中枢 — 决策引擎

基于智析数据 + 7 条决策规则，自动生成周度行动计划和优先级建议
